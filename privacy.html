Privacy Policy

Last updated: January 2026

1. Information Collected

This website collects minimal information necessary to provide a good user experience:

  • Contact Information: When you reach out via email, your name and email address are collected to respond to your inquiry.
  • Theme Preference: Your light/dark mode preference is stored locally in your browser (localStorage) and is never transmitted to any server.
  • Technical Data: Standard web server logs may include IP addresses, browser type, and pages visited for security and performance monitoring.

2. How Information is Used

Any information collected is used solely for:

  • Responding to your inquiries and communications
  • Improving website functionality and user experience
  • Ensuring website security and preventing abuse

Your information is never sold, rented, or shared with third parties for marketing purposes.

3. Data Storage and Security

This website is hosted on GitHub Pages. No personal data is stored on dedicated servers beyond standard web hosting logs. Theme preferences are stored only in your browser's localStorage.

All data transmission uses HTTPS encryption to protect information in transit.

4. Third-Party Services

This website may use the following third-party services:

  • GitHub Pages: Website hosting (subject to GitHub's Privacy Statement)

No advertising networks, tracking pixels, or social media widgets are used on this site.

5. Cookies

This website does not use cookies. Theme preferences are stored using localStorage, which is not transmitted with requests and cannot be used for tracking across sites.

6. Your Rights

You have the right to:

  • Request information about what data (if any) is held about you
  • Request deletion of any personal data
  • Clear your localStorage preferences at any time through your browser settings

For any privacy-related requests, contact jkindrix@gmail.com.

7. Changes to This Policy

This Privacy Policy may be updated periodically. Significant changes will be noted with an updated "Last updated" date at the top of this page.
